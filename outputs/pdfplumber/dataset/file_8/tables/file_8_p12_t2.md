| KISACrypto·V1.0은·보안요구사항을·적용받지·않는·구성요소가·존재한다(모든·구성요소·명세·필요).·각· 구성요소는·오용·시·검증대상·동작모드에·손상을·초래할·수·있는·SSP,·평문·데이터·또는·기타·정보를·처리하지· 않는다.· |  |
| 해설 |  |
| 보안요구사항을·적용받지·않는·모듈의·구성요소가·존재한다면,·각·구성요소가·오작동하거나·오용·시에도· 암호모듈의·검증대상·동작모드를·손상시키지·않는·근거를·명세한다. |  |
