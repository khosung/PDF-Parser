| 작성 예시 |  |
| KISACryptop·V1.0·암호모듈은·자가시험·실패에·대해·별도로·오류·상태를·출력하지·않는다.·각각의·자가· 시험을·실행하여·암호모듈이·모든·오류·상태에·진입하게·하여·암시적으로·오류·상태에·진입했음을·확인할·수· 있도록·하는·절차와,·그·때·암호·서비스가·수행·불가함을·확인할·수·있는·소스코드를·명세해야·한다. |  |
| 해설 |  |
| 암호모듈이·자가시험·실패에·대하여·오류·상태를·출력하지·않는다면,·암호모듈이·오류·상태에·진입했는지를· 결정할·수·있는·명백한·절차를·명세한다. |  |
