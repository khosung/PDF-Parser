| 보안요구사항 개요 | 암호모듈에·의해·생성된·프로세스에·대한·제한 |
| VE06.08.01 | 생성된·프로세스의·소유가·생성한·암호모듈로·국한시키는·방법·명세 |
| 작성 예시 |  |
| KISACrypto·V1.0·암호모듈은·범용·소프트웨어·라이브러리형·암호모듈이며,·암호모듈·동작·중에·추가·프로세스를· 생성하지·않는다.·암호모듈이·난수발생기의·잡음원·수집을·위해·사용하는·시스템·API는·운영체제가·제공하는· 기능이며,·암호모듈이·생성과·소멸·과정에·참여하지·않는다. |  |
| 해설 |  |
| 암호모듈이·추가·프로세스를·생성하는·경우,·생성되는·자식·프로세스의·소유권을·자식·프로세스를·생성한·부모· 프로세스로·국한하는·방법을·명세해야·한다.·프로세스를·생성하는·경우가·존재하지·않는다면·이·시험항목은· 프로세스를·생성하지·않는다고·명세하면·된다. |  |
